This will be a growing collection of 1000 hPa and 500 hPa Geopotential Height and 300 hPa Isotach fields since 1948. The 1000 hPa Geopotential Height field is a proxy for Mean Sea Level Pressure.

Dates will initially be gleaned from old Weather Guide Calendars. For every date that is entered, the 12z fields will be used. For significant events and multi-day events, I will include some OOz data and/or second and third day 12z data.

Eventually, SOM nodes will be added for the 500 hPa Geopotential Height and 300 hPa Isotach fields.
